Q: How do I start the server?
A: cd "e:\css - Copy" ; & ".\run.ps1"

Q: How do I access the dashboard?
A: http://127.0.0.1:5000/dashboard?user_id=1

Q: How do I stop the server?
A: Press Ctrl + C in the terminal

Q: How do I test the API?
A: cd backend ; python test_api.py

Q: Port 5000 is already in use?
A: Get-NetTCPConnection -LocalPort 5000 | Stop-Process -Force

Q: OpenAI API key error?
A: Check .env file exists and contains OPENAI_API_KEY=sk-proj-...

Q: Database error?
A: Run: python -c "from app.models.models import db; from main import app; app.app_context().push(); db.create_all()"

Q: ModuleNotFoundError?
A: Activate venv and run: pip install flask flask-cors sqlalchemy openai
